# RAWTALK

RAWTALK is a simple English speaking practice app built to help me improve daily conversation, pronunciation, grammar and job interview confidence.

The goal is simple: speak every day, get corrections, listen, repeat and improve.

## Features

- Daily English practice sessions
- Speech recognition
- Text-to-speech
- Gemini AI English coach
- Grammar correction
- Follow-up questions
- Practice progress
- Interview-focused English prompts
- Simple UI, no unnecessary complexity

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Gemini API
- Web Speech API

## Main Flow

```txt
Question
↓
User speaks
↓
Speech recognition converts voice to text
↓
Gemini analyzes the answer
↓
Coach gives correction, tip and follow-up question
↓
User keeps practicing
